HTML5

A través de varios capítulos vamos a descubrir el principal lenguaje utilizado para la creación de páginas web: Hyper Text Markup Language, más conocido mediante sus siglas HTML

El HTML es un "lenguaje de marcado". Basa su sintaxis marcas, tags o etiquetas.
CSS3

El CSS, es el segundo lenguaje más básico y esencial para crear páginas web; con él se define la parte de la presentación (cómo mostrar los elementos de la página, su posición, forma, espaciados, colores; toda la parte estética.

El enfoque de CSS es servir para definir la capa de presentación; la parte relacionada con el aspecto.
Bootstrap

Boostrap es un Framework que permite generar en forma más sencilla y con menos código un FrontEnd de una página web en forma responsive y basándose en las cualidades de CSS.

Antes de utilizar Bootstrap, asegúrate de que tus páginas utilizan el doctype de HTML5
